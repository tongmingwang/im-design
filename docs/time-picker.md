# TimePicker 时间选择

时间选择器，用于选择时间。

## 基本用法

<ImTimePicker   />

```vue
<template>
  <ImTimePicker />
</template>
```

## 支持清除

设置 `clearable` 属性，可以显示清除按钮。

<ImTimePicker  clearable />

```vue
<template>
  <ImTimePicker clearable />
</template>
```

## 禁用

设置 `disabled` 属性，可以禁用时间选择器。

<ImTimePicker  disabled />

```vue
<template>
  <ImTimePicker disabled />
</template>
```
